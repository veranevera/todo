# ToDo
Simple ToDo list application with React and Redux.
# Installation

1. Clone the repository: git clone https://github.com/veranevera/todo.git
2. Write file with global environment variables: npm run setup
3. Change value of variables in .env file
4. Install the application: npm install

Development

1. Start webpackDevServer: npm run dev
2. View in browser at http://localhost:8080

Production

1. Build souces: npm run build

# Tests
To start watching tests: npm run test:watch
